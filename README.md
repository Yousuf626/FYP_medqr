# FYP_medqr
In the healthcare landscape, managing medical records faces challenges of data security, accessibility, and patient-doctor interactions. Traditional methods yield data vulnerabilities, medication errors, and hinder communication. MedQR is a block chain-powered mobile app which redefines healthcare record management, empowering patients to securely manage medical data, prescriptions, and medication schedules. It provides doctors swift access through QR codes. Combining block chain security, mobile app flexibility, and user-friendliness, MedQR promises to revolutionize healthcare. It ensures data integrity, boosts patient engagement, and promotes treatment plan adherence. MedQR is poised to elevate patient care, streamline processes, and enhance healthcare's overall quality.
